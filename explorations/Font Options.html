Typeface options for Navric

Three pairings — a neutral sans + a tabular mono — each shown on a heading, a verdict, body copy, and the numerics that fill this product (score, IDs, rates, deltas, a right-aligned column to test figure alignment). Numbers use the mono with tabular-nums.

All three are free (Google Fonts), so they load the same way Geist does now — no licensing step. My pick is Geist + Geist Mono: it's the closest to the Linear/Sana north star, genuinely neutral so severity color stays the loudest thing on screen, and Geist Mono's tabular figures are excellent for dense fleet data. IBM Plex reads more "engineered/compliance" if you want extra gravity; Hanken + JetBrains is a touch warmer and more distinctive.
