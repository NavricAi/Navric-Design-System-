Accent options for Navric

The same UI cluster — primary button, active nav, link, watchlist pill, focus ring — under four accent directions. A constant Severe badge sits in each so you can judge how the accent coexists with risk color. Severity must always win the eye.

Why these: your severity ramp owns green + amber + orange + red, and teal is reserved for "improving." That leaves only the cool/blue family as conflict-free chromatic room — so the cleanest way to drop blue/purple is to go achromatic (ink). My pick is Navy ink: it reuses a brand color you already own (#0B1437), competes with nothing, and lets severity be the only hue on the screen.
